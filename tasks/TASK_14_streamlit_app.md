# TASK-14: Streamlit Visualization App

## Goal
Implement the Streamlit app for interactive visualization of DESeq2 results, GSEA pathway analysis, and QC metrics. App reads pre-generated data files only; no pipeline DB write access.

## Requirements
- Pages: QC Dashboard, Differential Expression (volcano plot, MA plot, heatmap), Pathway Enrichment (bubble chart, table), Single-Cell (UMAP, marker genes).
- Data loading: reads Streamlit-ready files generated by `prepare_streamlit_data` tool.
- Filters: padj cutoff slider, LFC cutoff slider, contrast selector, top-N pathway selector.
- Plots: Plotly (interactive), with PNG export button per plot.
- No DB connection; reads from `streamlit_data/` directory mounted into container.
- `Dockerfile.streamlit` already created in TASK-01; this task implements the app content.

## Acceptance Criteria

- [ ] MA plot renders for fixture DE data.
